# Board icon pack

Extra SVG board icons for Latch. After enable, open **Admin → Boards** and pick an icon for each board (or rely on name/slug auto-suggest for keys like `docs`, `security`, `server`).

## Install

- **Admin → Plugins → Catalog** (when published), or  
- `php bin/latch plugin install /path/to/board-icon-pack` then enable.
